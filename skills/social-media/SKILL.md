## 各平台規格與最佳實踐

### Instagram (IG)

**帳號定位**：品牌知識傳播 + 社群情感連結的核心主場
**受眾組成**：30-50 歲耐力運動員，Garmin/COROS 使用者，賽事跑者

#### 貼文（Feed Post）
| 項目 | 規格/建議 |
|------|---------|
| 圖片尺寸 | 1:1（1080×1080px）或 4:5（1080×1350px）— 後者在動態中佔版面更大 |
| 影片長度 | 最長 60 秒（建議 15-30 秒，前 3 秒必有 Hook） |
| 字數 | 主文 150-300 字，前 3 行必須有 Hook（展開前可見） |
| Hashtag 數量 | 10-15 個（放在主文末尾或第一則留言） |
| 發布頻率 | 每週 2-3 則 |
| 最佳發布時間 | 週間：06:30-08:30、12:00-13:00、20:00-22:00；週末：08:00-10:00 |

#### Carousel（輪播）
- 最多 10 張，建議 3-6 張（奇數張效果較佳）
- 第一張必須是強力封面（決定是否滑動）
- 每張文字不超過 3 行（橫幅式）
- 最後一張放 CTA + 品牌 Logo

#### Stories（限時動態）
| 項目 | 規格/建議 |
|------|---------|
| 尺寸 | 9:16（1080×1920px）|
| 文字最小尺寸 | 16pt 以上（手機可讀） |
| 發布頻率 | 每日 1-2 則 |
| 使用功能 | 投票貼圖、問答貼圖、倒數貼圖（活動時） |
| 有效窗口 | 24 小時 |

#### Reels
- 最佳長度：15-30 秒（知識型）、60-90 秒（賽事故事型）
- 前 3 秒視覺/文字 Hook 決定留存率
- 必加字幕（靜音播放環境）
- 結尾畫面：品牌 Logo + 追蹤 CTA，停留 3 秒

### Facebook (FB)

**帳號定位**：品牌知識深度傳播 + 老朋友互動空間
**受眾組成**：35-55 歲，長期追蹤品牌的資深跑者，跑團社群滲透

| 項目 | 規格/建議 |
|------|---------|
| 圖片尺寸 | 1:1（最兼容）或 1.91:1（橫版連結圖） |
| 字數 | 可承載 200-400 字，適合條列式重點 |
| Hashtag 數量 | 5-8 個（FB 不需要太多 Hashtag） |
| 發布頻率 | 每週 1-2 則 |
| 互動設計 | 每則貼文結尾必須有引導留言的問句 |
| 社團策略 | 主動在跑者相關社團（台灣跑者、鐵人三項同好）分享知識內容 |
| 廣告整合 | FB 貼文可直接作為廣告素材再投放 |

### LINE Official Account（LINE OA）

**帳號定位**：最私密的品牌溝通管道，一對一服務感
**受眾組成**：主動加入 LINE 好友的忠誠客戶與高意向潛在客戶

| 項目 | 規格/建議 |
|------|---------|
| 圖文訊息長度 | 不超過三屏手機畫面 |
| 語氣設定 | 最口語化，像好朋友傳訊息 |
| 發送頻率 | 每週 1-2 則（不過度打擾） |
| 強調格式 | 使用【 】或 ▶ 標注重要資訊 |
| CTA | 每則必須有明確下一步（連結按鈕或回覆指令） |
| 不使用 | Hashtag（LINE 平台不適用）|
| 圖文卡片 | 主要促銷/活動使用圖文卡片格式，視覺吸引力強 |
| 自動回覆 | 設定常見問題自動回覆（如輸入「眠易復」自動回傳產品資訊） |

### 官網 / 部落格

**定位**：SEO 流量主力，品牌知識權威中心
- 文章長度：1,500-3,000 字
- H1/H2 標題含目標關鍵字
- 每篇文章結尾連結至相關產品頁

### 蝦皮 / 電商平台

**定位**：銷售轉換最前線，資訊清晰第一
- 主圖：產品正面白底 + 核心賣點文字
- 圖文說明：規格 → 成分 → 建議用法 → 常見問題
- 語氣：簡潔直接，規格清楚
- 賣點標籤：「SGS 檢測通過」「NTSU 國體大實證」「全素可食」

**第一層：品牌標籤（固定使用）**
```

#堅心
#OutrunNutrition

```

**第二層：功能標籤（依內容主題選擇 3-5 個）**
```
恢復類：#運動恢復 #科學恢復 #深度睡眠 #睡眠品質
補給類：#科學補給 #運動補給 #恢復蛋白 #運動益生菌
訓練類：#耐力訓練 #訓練恢復 #HRV #Garmin
```

**第三層：賽事標籤（依當前賽事季節選擇 2-4 個）**
```
馬拉松：#馬拉松 #全馬 #半馬 #跑步 #路跑
鐵人：#鐵人三項 #鐵人 #Triathlon #鐵人226 #鐵人113

綜合：#跑者 #台灣跑者 #耐力運動
```

### Hashtag 選用原則
- IG 貼文：總計 10-15 個，三層各取適量
- 避免使用封鎖 Hashtag（定期確認 IG 政策）
- 品牌標籤固定出現，其他依主題調整
- 賽事季節對應換賽事 Hashtag（如馬拉松旺季強化跑步標籤）

## 社群互動 SOP

### 日常互動流程

**每日必做（10 分鐘）**
1. 查看所有平台的新留言、標記
2. 回覆所有留言（優先 24 小時內）
3. 查看 Hashtag 上的相關貼文，點讚互動
4. 限時動態更新（恢復打卡 or 短知識）

**每週必做（30 分鐘）**
1. 規劃當週貼文（預排 IG/FB 2-3 則）
2. LINE OA 訊息準備
3. 查看上週貼文數據，標記高互動內容類型
4. 更新自動回覆設定（如有新活動）

### 留言互動原則

**一般互動留言**
- 24 小時內回覆
- 先肯定對方（例如：「謝謝你的支持！」「今天的長距離辛苦了！」）
- 回答完問題後鼓勵進一步互動

**問題類留言**
- 簡短問題：直接在留言回答
- 複雜問題：「這個問題很好！私訊我們，我們詳細說明」
- 比較複雜的個人化建議：引導私訊，一對一深入說明

**批評/負面留言**
→ 詳見「負面留言處理流程」

## 負面留言處理流程

### 分類處理決策樹

```
收到負面留言
     ↓
是否為真實的使用者反饋？
     ↓              ↓
    YES             NO（廣告機器人/無理取鬧）
     ↓              ↓
是否涉及產品問題？    不處理，可隱藏惡意內容
     ↓      ↓
    YES     NO（情緒性抱怨）
     ↓      ↓
公開回應 +  先表達同理，
私訊跟進   再了解具體情況
```

### 各類型標準回應

**類型一：產品效果不如預期**
公開回應原則：先同理 → 說明個人差異 → 邀請私訊深入了解
> 「謝謝你的真實分享！每個人的訓練狀況和身體反應不同，我們很想更了解你的使用情況。能私訊我們嗎？我們想確認使用時機是否有可調整的空間 🙏」

**類型二：價格疑慮**
公開回應原則：不辯護 → 說明成本背後的價值 → 提供入門方案
> 「我們理解補給品的預算是很實際的考量。堅心的成本包含 NTSU 研究投入和 SGS 安全檢測，這是我們不妥協的底線。如果想先試試看，能易復的 500g 盒裝是個好起點 😊」

**類型三：誤導性資訊/不當指控**
公開回應原則：客觀澄清 → 提供可查證資訊 → 不情緒化
> 「關於這個部分，我們想做個說明：[客觀澄清]。詳細的成分說明和 SGS 檢測報告可在官網查閱。有任何疑問歡迎私訊我們 ✅」

**類型四：物流/購買問題**
公開回應：立即道歉 + 引導私訊 + 承諾解決
> 「非常抱歉讓你有這樣的體驗！請立即私訊我們並附上訂單編號，我們會優先為你處理 🙏」

### 私訊跟進清單
- [ ] 24 小時內完成公開回應
- [ ] 私訊跟進詳細了解問題
- [ ] 提供解決方案或補償（視情況）
- [ ] 記錄案例，更新 FAQ 內容庫

## 素材風格規範

### 整體視覺調性關鍵詞
- 高品質、寫實感、微光攝影
- 暖色調、自然光、汗水感
- 留白呼吸感、生活化但有質感

### 色彩應用規範
| 場景 | 色彩應用 |
|------|---------|
| 貼文底色 | 堅心黑 `#2F2F2F`（主）或自然光攝影圖（輔） |
| 文字 | 柔和白 `#E0E0E0`（主文）、中性灰 `#A0A0A0`（次要） |
| 強調重點 | 堅心橘 `#ED5D2B`（最多佔 10%）|
| 科學標籤 | 堅心藍 `#145DDD`（NTSU/成分標籤） |

### 圖片選用原則
✅ 適合使用：
- 真實運動場景（汗水感）
- 自然光 / 微光攝影
- 生活化構圖（早晨、訓練後）
- 清晰的產品特寫（白底或黑底）

❌ 避免使用：
- 攝影棚打光感（過於商業化）
- 過度精修、無瑕疵的完美照片
- 誇張表情或動作的圖庫素材
- 純黑 `#000000` 背景
- 圖片上大量文字（超過畫面 20%）

### 限時動態素材指引
- 背景：從堅心黑到階層深灰的微漸層
- 文字大小：最小 16pt
- 互動貼圖：活動期間善用倒數/投票/問答
- 節奏：交替使用純色底與真實照片，避免連續深色

### 輪播貼文節奏
- 多圖輪播交替：深色底文字圖 ↔ 真實運動照片
- 創造明暗節奏，避免連續純深色造成沉悶感
- 最後一張設計：品牌 Logo + CTA 按鈕風格
