## 文章 SEO 結構規範

### Title Tag 規範
- 格式：`[主要關鍵字] | [次要關鍵字]｜堅心 Outrun Nutrition`
- 字元數：50-60 字元（英文計算）
- 範例：`馬拉松賽後恢復飲食指南：黃金30分鐘補給策略 | 堅心 Outrun Nutrition`

### Meta Description 規範
- 字元數：150-160 字元
- 必須包含：主要關鍵字 + 內容摘要 + 行動呼籲
- 範例：`馬拉松賽後最關鍵的30分鐘，碳水：蛋白質=1.2:1才是科學恢復比例。堅心Outrun Nutrition提供NTSU實證的耐力運動員恢復指南，讓每次訓練的努力確實轉化為進步。`

### URL Slug 規範
- 使用英文（有利於國際 SEO）
- 使用連字符（-）而非底線（_）
- 長度：3-5 個單詞
- 範例：`/blog/marathon-recovery-nutrition-guide`

### H1-H6 標題結構

```
H1（每頁唯一）：主要關鍵字 + 吸引點擊的元素
├── H2（2-5 個）：次要關鍵字，分段主題
│   ├── H3（每個 H2 下 2-4 個）：長尾關鍵字，細節說明
│   └── H3
└── H2
    └── H3（FAQ 段落使用）
```

### 文章內連結策略

**內部連結原則（每篇文章 3-5 個）**
- 連結至相關的支柱頁面（如從 Cluster 文章連至 Pillar 頁面）
- 連結至相關產品頁（自然帶入，非硬塞）
- 使用描述性錨文字（Anchor Text）而非「點擊這裡」
- 錨文字範例：「了解更多眠易復的深度睡眠研究」

**外部連結原則（每篇文章 2-3 個）**
- 優先連結至 PubMed 研究論文（提升可信度）
- NTSU 官方網站
- ACSM（美國運動醫學會）等權威機構
- 不連結至競品網站

## 內容權威性規範（引用來源優先順序）

### 第一優先：同儕審查學術期刊
- **可信度**：最高
- **適用場景**：成分機制說明、功效數據引用
- **常用來源**：
  - PubMed（https://pubmed.ncbi.nlm.nih.gov/）
  - ISSN（國際運動營養學會）
  - Journal of Sports Sciences
  - European Journal of Sports Science
- **引用格式**：
  ```
  作者姓名（發表年份）。〈文章標題〉。《期刊名稱》，卷期，頁碼。
  DOI：https://doi.org/...
  ```

### 第二優先：NTSU 國立體育大學實證數據
- **可信度**：高，且為堅心獨特競爭優勢
- **適用場景**：產品功效說明、品牌信任建立
- **使用原則**：引用前確認數據版本和授權範圍

### 第三優先：權威運動科學組織官方指南
- ACSM（美國運動醫學會）：https://www.acsm.org/
- NSCA（國家肌力與體能訓練協會）
- 台灣運動醫學學會

### 第四優先：具執照的專業人士臨床意見
- 必須標明「持照營養師審核」或「運動科學顧問 審核」
- 不得引用未具資格的網絡意見

### 數據呈現規範
- 優先使用量化指標（HRV 變異率、恢復時間縮短百分比）
- 引用數據時同步說明實驗背景（樣本數、研究設計）
- 若引用研究數字，必須說明「針對 [樣本描述] 的 [研究設計] 顯示...」

## FAQ Schema 建置指南

FAQ（常見問題）段落是獲取 Google 精選摘要（Featured Snippet）的有效手段。

### FAQ Schema 格式

每篇文章的 FAQ 段落建議包含 3-5 個問題：

```html
<!-- 在文章中使用結構化格式標記 -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "問題內容",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "回答內容"
      }
    }
  ]
}
</script>
```

### FAQ 問題撰寫原則
- 使用搜尋者實際會輸入的問句形式
- 回答簡潔（50-100 字），但完整
- 回答中自然融入目標關鍵字
- 問題從長尾關鍵字中挑選（搜尋量有依據）

### 常用 FAQ 問題模板

**產品類**
- 「眠易復適合什麼樣的運動員使用？」
- 「能易復一天可以喝幾次？」
- 「食動力要在運動前多久服用？」

**科學類**
- 「什麼是 HRV？對跑者有什麼意義？」
- 「耐力運動後為什麼需要補充碳水化合物？」
- 「TWK-10® 益生菌和一般益生菌有什麼不同？」

**購買類**
- 「堅心的產品通過什麼安全認證？」
- 「眠易復可以長期服用嗎？」
- 「堅心的產品素食者可以食用嗎？」

- [ ] Google Search Console：確認無抓取錯誤，檢視點擊率低的頁面

- [ ] Core Web Vitals：LCP < 2.5s、FID < 100ms、CLS < 0.1
- [ ] 網站速度：行動版 PageSpeed 分數 > 70

- [ ] 關鍵字排名追蹤（目標關鍵字位置變化）
- [ ] 競品 SEO 動態（是否推出類似文章）
