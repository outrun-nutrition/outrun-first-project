## 四大堅心專屬寫作公式

### 公式一：【科學開箱】— 數據共感化公式

**適用系列**：科學開箱（Scientific Unboxing）
**核心邏輯**：現象 → 數據原因 → 體感解方

**公式結構：**
```
[現象描述]：你是否也有過 [具體、真實的運動困擾場景]？

[數據背書]：研究顯示，這不是你的問題，而是 [科學機制解釋]。
             根據 [來源/數據]，[量化指標說明]。

[體感解方]：這就是為什麼 [成分/產品] 的設計是 [簡單的解決邏輯]。
             換成你的體感：[數據共感化描述]。

[行動呼籲]：[知識延伸 / 產品了解 / 互動問句]
```

**範例：**
> 練完反而更睡不著？這不是你不夠累，是你的交感神經還在加班。
>
> 研究顯示，高強度耐力訓練後，交感神經系統平均需要 2-4 小時才能恢復至靜息狀態。你睡不著，不是因為你不累，而是身體還卡在「戰鬥模式」裡。
>
> 眠易復的檸檬馬鞭草成分，就是針對這個「切換機制」設計的。換成你的體感：就是「訓練完回家終於可以真正關機，而不是躺著繼續高速運轉」。
>
> 你的 Garmin 深層睡眠時間通常是幾分鐘？留言告訴我們 👇

### 公式二：【生活平衡】— 場景共鳴公式

**適用系列**：生活平衡（Life Balance）
**核心邏輯**：真實場景 → 情感共鳴 → 支持承諾

**公式結構：**
```
[場景描述]：[具體時間點] + [角色狀態] + [感受描述]

[情感共鳴]：我們懂 [撕裂感/矛盾感/疲憊感]。
             這不是軟弱，這是 [重新框架，換個角度]。

[品牌支持]：補給，不是為了 [錯誤的目的]，
             而是為了 [真正的目的]。

[暖心收尾]：[品牌人格的表達] + [輕度 CTA 或互動]
```

**範例：**
> 清晨 5 點的鬧鐘響了，你換上跑鞋出門。回來後要開 9 點的會議，要陪孩子吃早餐，要準備今天的簡報。
>
> 我們懂那種「訓練完卻沒有體力陪家人」的撕裂感。這不是你不夠愛他們，是你把一天的力氣都給了賽道。
>
> 補給，不是讓你跑更快，而是讓你跑完還有餘力做自己想做的事。
>
> 堅心在，你不用在「運動員」和「父母/主管/夥伴」之間選邊站 🙌

### 公式三：【實戰指南】— SOP 斷言公式

**適用系列**：實戰指南（Action Guide）
**核心邏輯**：結果承諾 → 步驟清單 → 行動呼籲

**公式結構：**
```
[結果承諾]：[動詞] + [目標] + [時間/條件框架]

步驟 1：[簡潔動詞] + [具體行動]
步驟 2：[簡潔動詞] + [具體行動]
步驟 3：[簡潔動詞] + [具體行動]
（3-5 步驟最佳，不超過 7 步）

[補充說明]：[最關鍵的執行提示，1-2 句]

[行動呼籲]：[儲存這篇 / 查看完整指南 / 詢問客製方案]
```

**範例：**
> 賽後 30 分鐘黃金恢復期，你只需要做這 3 件事：

> 1. **補碳水先**：1 包能易復，溫水沖開，在終點線旁就喝下去
> 2. **動態舒緩**：不要直接坐下，慢走 5-10 分鐘讓血液循環
> 3. **睡前關機**：當天晚上服用眠易復，讓深層睡眠完成最後修復
>
> 關鍵：不要等「回到飯店再說」——賽後 30 分鐘是補給吸收效率最高的窗口期。
>
> 儲存這篇備用，下次比賽對照執行 ⚡

### 公式四：【透明研發】— 幕後誠實公式

**適用系列**：透明研發（Behind the Brand）
**核心邏輯**：問題起點 → 研發過程 → 承諾宣言

**公式結構：**
```
[問題起點]：我們在開發 [產品] 時，一直在想一個問題：
             [開發初衷的核心疑問]

[研發過程]：我們嘗試過 [方向一]，但 [問題]。
             後來 [轉折點]，才找到 [解決方向]。

[結果說明]：這就是為什麼 [產品] 裡有 [成分/特色]，
             不是因為它便宜，而是因為它 [真正的原因]。

[承諾宣言]：我們不做 [負面對比]，我們只做 [正面承諾]。
```

**範例：**
> 我們在開發能易復的口味時，一直在想一個問題：為什麼運動補給品都要像藥水？
>
> 我們嘗試過各種「安全的商業口味」，但發現都有一種讓人感覺在「吞藥」的距離感。後來我們問了一百多位台灣跑者：「跑完 42 公里後，你最想要的味道是什麼？」
>
> 黑糖仙草。
>
> 這就是為什麼能易復是黑糖仙草——不是因為它最容易做，而是因為這個味道對台灣跑者來說，帶著一種「完成了，好好獎勵自己」的心理意義。
>
> 補給的最後一哩路，是讓你願意每次都喝下去。

## 標題寫作技巧

### 六種有效標題公式

**公式 1：數字清單型**
> 「[數字] 個你可能不知道的 [主題] 關鍵」
> 範例：「3 個全馬跑者最常犯的恢復錯誤」

**公式 2：疑問挑戰型**
> 「為什麼你 [行為] 卻 [反常結果]？」
> 範例：「為什麼你睡了 8 小時，手錶還是亮紅字？」

**公式 3：直接利益型**
> 「[時間框架] 內，[具體可見的結果]」
> 範例：「賽後 30 分鐘，讓雙腿明天還能動的關鍵」

**公式 4：對比反差型**
> 「不是 [常見錯誤觀念]，而是 [正確觀念]」
> 範例：「不是練得不夠多，而是恢復做得不夠好」

**公式 5：場景代入型**
> 「[具體場景]，你是否也 [情感共鳴]？」
> 範例：「訓練完回家，卻發現自己對家人沒有耐心——你不孤單」

**公式 6：權威數據型**
> 「[百分比/數字] 的 [目標族群] 不知道的 [主題] 事實」
> 範例：「超過 60% 的業餘耐力運動員有過度訓練的早期徵兆」

## Hook 鉤子模板庫

好的 Hook 決定前三秒，前三行。

### 數據衝擊型
- 「[數字]%的耐力運動員[令人驚訝的統計]——你是那個嗎？」
- 「研究顯示，[常見行為]實際上讓恢復效率下降[數字]%。」

### 場景代入型
- 「清晨 5 點的鬧鐘響了。你換上跑鞋，準備迎接今天的[距離]...」
- 「訓練完的你，最想做的第一件事是什麼？」

### 問題挑戰型
- 「如果我說，你可能一直把恢復做錯了——你願意聽嗎？」
- 「你有多久沒有在 Garmin 上看到全綠色的恢復分數了？」

### 反直覺型
- 「練得越多，進步越少——這不是悖論，這是生理學。」
- 「你最需要的補給，不是在訓練中，而是在訓練後的 30 分鐘。」

## 多版本產出規範

### 兩版本標準
每次創作內容時，預設提供：

**專業穩重版**
- 語氣：理性、科學感強、具備數據佐證
- 受眾：核心運動員、關注成分科學的消費者
- 強調：NTSU 實證、成分機制、量化指標

**感性共鳴版**
- 語氣：溫暖、場景感強、情感連結深
- 受眾：初次接觸品牌、被故事打動的消費者
- 強調：生活場景、真實痛點、關係共鳴

### 版本選用建議
| 情境 | 建議版本 |
|------|---------|
| 官網產品頁 | 專業穩重版（或兩版融合） |
| IG 知識型貼文 | 兩版均適用，依當週主題選擇 |
| FB 廣告文案 | 感性共鳴版作為主力 |
| LINE OA | 感性共鳴版，再口語化 |
| KOL 合作文案 | 感性共鳴版（讓 KOL 加個人化調整） |

## 品質檢查清單

每篇內容發布前：

**內容品質**
- [ ] 前三行/句有強力 Hook
- [ ] 有清楚的段落結構（不是大段文字）
- [ ] 有明確的行動呼籲（CTA）
- [ ] 字數符合平台規範

**品牌聲音**
- [ ] 使用「你」而非「您」
- [ ] 語氣符合當前通路設定
- [ ] 數據已做共感化轉換
- [ ] 無否定讀者的措辭

**法規合規**
- [ ] 無醫療效能聲稱
- [ ] 功效描述使用合規語境
- [ ] 無誇大不實用語

**平台格式**
- [ ] 符合各平台字數/格式規範
- [ ] Hashtag 數量適當
- [ ] 視覺設計建議已附上
