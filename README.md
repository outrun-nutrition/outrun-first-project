# 運動營養計算器 — Cyberbiz 嵌入工具

一個互動式的運動營養需求計算器，根據使用者的身體數據與訓練目標，計算每日營養素需求並推薦適合的運動保健品。

## 功能特色

- **4 步驟互動表單**：基本資料 → 訓練頻率 → 訓練目標 → 個人化結果
- **營養需求計算**：基於 Mifflin-St Jeor 公式，結合訓練強度與目標計算 TDEE 與巨量營養素
- **智慧補充品推薦**：依據運動類型、目標、困擾自動推薦對應保健品（乳清蛋白、肌酸、BCAA、電解質等）
- **響應式設計**：手機、平板、桌機均可正常顯示
- **繁體中文介面**：完整中文化，適合台灣市場

## 如何嵌入到 Cyberbiz

### 方法一：自訂頁面（推薦）

1. 進入 Cyberbiz 後台 → **網站管理** → **頁面管理** → **新增頁面**
2. 選擇 **自訂 HTML** 編輯模式
3. 將 `nutrition-calculator.html` 的完整內容貼入
4. 儲存並發布頁面

### 方法二：嵌入現有頁面

1. 進入想嵌入的頁面編輯器
2. 新增一個 **自訂 HTML 區塊**
3. 將 `nutrition-calculator.html` 中 `<style>...</style>` 和 `<div id="ncApp">...</div>` 及 `<script>...</script>` 部分貼入
4. 儲存頁面

## 計算邏輯說明

| 項目 | 公式/依據 |
|------|-----------|
| BMR | Mifflin-St Jeor 公式 |
| TDEE | BMR × 活動係數（1.375 ~ 1.9） |
| 蛋白質 | 體重 × 1.6~2.4g（依目標與訓練頻率） |
| 碳水化合物 | 依 TDEE 剩餘熱量分配 |
| 脂肪 | TDEE 的 20~30%（依目標調整） |

## 推薦產品類別

- 乳清蛋白、肌酸、BCAA、電解質補充飲
- 麩醯胺酸、魚油、訓練前補充
- 鎂、維生素 D3、咖啡因錠

## 客製化

若需要調整品牌色系或連結到特定商品頁面，可修改以下部分：

- **色系**：修改 CSS `:root` 中的 `--primary` 等變數
- **商品連結**：在 `buildRecommendations()` 函數中為產品新增 `url` 欄位
