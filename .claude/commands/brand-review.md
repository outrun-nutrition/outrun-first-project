## 審查面向一：品牌聲音檢查

### 核心語氣標準
堅心的品牌語氣定位：「暖感科學（Warm Science）」— 理性的框架，感性的內容

**語氣關鍵詞（應有）**
- 親近感：使用「你」而非「您」
- 同理心：先肯定對方的努力，再提建議
- 科學感：有數據佐證，但數據要轉化為體感描述
- 誠實感：說真實情況，不誇大

**禁用詞庫**
| 禁用詞 | 原因 | 建議替代詞 |
|--------|------|-----------|
| 保證 | 誇大承諾 | 根據實測數據顯示 |
| 神效 | 誇大不實 | 顯著提升 / 有效輔助 |
| 奇蹟 | 誇大不實 | 科學驗證 |
| 立刻見效 | 不實聲稱 | 連續使用後 |
| 全網最低 | 不當比較 | 刪除 |
| 絕對有效 | 誇大不實 | 臨床實測顯示 |
| 神級 | 誇大詞彙 | 卓越的 / 專業級的 |
| 你做錯了 | 否定讀者 | 我們可以做得更好 |
| 你不懂 | 否定讀者 | 試試看另一種方式 |
| 最強 / 最好 | 不當比較 | 專為耐力運動員設計 |

**品牌聲音審查清單**
- [ ] 使用「你」而非「您」
- [ ] 無否定讀者的措辭
- [ ] 無過度誇大的行銷用語
- [ ] Emoji 用量適中（每段最多 1-2 個）
- [ ] 句子簡短有力，適合手機閱讀
- [ ] 扮演「資深跑友」角色，非高高在上的指導者

## 審查面向二：法規合規檢查（食安法 §28）

### 核心法規說明
根據台灣《食品安全衛生管理法》第 28 條，食品廣告不得宣稱醫療效能，不得有虛偽、誇大、易生誤解之情形。

### 高風險詞彙對照表

| ❌ 違規用語 | ✅ 合規替代 | 說明 |
|-----------|-----------|------|
| 治療失眠 | 優化睡眠品質 / 輔助改善睡眠 | 治療屬醫療效能 |
| 治癒疲勞 | 幫助恢復體力 / 輔助緩解疲勞感 | 治癒屬醫療聲稱 |
| 預防受傷 | 有助維持正常生理機能 | 預防疾病屬醫療範疇 |
| 消炎止痛 | 輔助維持運動後正常生理恢復 | 醫療效能 |
| 提高免疫力 | 維持免疫系統正常運作 | 需有科學依據且用語謹慎 |
| 排毒 | 刪除，無科學根據 | 不當聲稱 |
| 燃燒脂肪 | 有助於維持體重管理計畫 | 誇大效能 |
| 快速恢復 | 有助縮短恢復期 / 輔助恢復 | 「快速」暗示保證效果 |
| 100% 有效 | 根據實測數據顯示 __% 受測者... | 絕對聲稱 |
| 副作用為零 | 天然成分、SGS 安全檢測通過 | 不得承諾零副作用 |
| 根治 | 刪除 | 醫療效能 |

### 合規語境模板
功效描述時使用以下安全語境：
- 「本產品提供 __ 營養素，有助於維持 __ 的正常生理機能」
- 「根據 __ 實驗數據，連續使用 __ 週後，受測者的 __ 指標顯示 __」
- 「輔助運動後恢復，幫助維持正常的肌肉機能」
- 「提供身體所需的 __，補充運動消耗的營養」

**法規合規審查清單**
- [ ] 無「治療」、「治癒」、「預防疾病」等醫療效能詞彙
- [ ] 無虛偽不實、誇大或易生誤解之聲稱
- [ ] 若提及功效數據，已說明實驗背景（樣本數、測試時長）
- [ ] 功效描述使用「輔助」、「維持正常生理機能」等合規語境
- [ ] 無「100% 有效」等絕對性聲稱
