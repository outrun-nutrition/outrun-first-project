# 社群貼文撰寫流程

## 指令說明
根據堅心 Outrun Nutrition 的品牌規範，撰寫適用於 IG、FB、LINE OA 三個平台的社群貼文。每篇自動產出「專業穩重版」與「感性共鳴版」兩個版本。

---

## 執行步驟

### STEP 1：確認貼文需求

請提供以下資訊：
- **平台**：IG / FB / LINE OA（或多平台同步）
- **內容系列**：科學開箱 / 生活平衡 / 實戰指南 / 透明研發
- **主題方向**：（例如：眠易復產品介紹 / 賽後恢復技巧 / HRV 數據解讀）
- **目標行動**（CTA）：購買連結 / 互動留言 / 知識分享 / 預約諮詢

---

### STEP 2：選擇內容系列風格

#### 📊 【科學開箱】系列
- **語氣**：理性、邏輯清晰、數據佐證
- **結構**：現象 → 數據原因 → 解決方案
- **Hook 公式**：「為什麼 [常見困惑]？這不是你的問題，而是 [科學原因]。」
- **適用主題**：成分解析、實驗數據公開、補給時機說明

#### 💛 【生活平衡】系列
- **語氣**：溫暖、同理、感性連結
- **結構**：具體生活場景 → 情感共鳴 → 解決方案
- **Hook 公式**：「那種 [具體疲憊場景] 的感覺，你懂嗎？」
- **適用主題**：斜槓運動員的日常、訓練與家庭平衡、疲勞轉換

#### ⚡ 【實戰指南】系列
- **語氣**：簡潔、斷言、行動導向
- **結構**：直接給 SOP，不拖泥帶水
- **Hook 公式**：「[數字] 個步驟，讓你的 [目標] 更有效率：」
- **適用主題**：賽事補給 SOP、恢復流程、訓練前後的補給時機

#### 🔬 【透明研發】系列
- **語氣**：誠實、開放、去神祕化
- **結構**：開發背後的故事 → 與其他產品的本質差異 → 品牌承諾
- **Hook 公式**：「我們為什麼選擇 [成分/口味]？答案可能跟你想的不一樣。」
- **適用主題**：成分選材理由、SGS 檢測透明、口味開發故事

---

### STEP 3：各平台撰寫規範

#### Instagram (IG)
- **字數**：主文 150-300 字，前三行必須有 Hook
- **段落**：每段 2-3 行，大量換行增加閱讀性
- **Emoji**：適量使用，每段 1-2 個，增加視覺節奏
- **Hashtag 規範**（放在貼文末尾）：
  - 品牌標籤：`#堅心OutrunNutrition` `#堅心` `#OutrunNutrition`
  - 功能標籤：`#運動恢復` `#科學補給` `#深度睡眠` `#耐力運動`
  - 賽事標籤：`#馬拉松` `#鐵人三項` `#跑步` `#全馬`
  - 總數：10-15 個
- **CTA**：置於貼文倒數第二段，行動呼籲明確

#### Facebook (FB)
- **字數**：主文 200-400 字，可承載更多資訊
- **連結**：可直接放產品或文章連結
- **格式**：適合條列式重點（3-5 點）
- **互動設計**：結尾加入問句引導留言（如：「你通常在賽後多久補給？留言告訴我們！」）
- **Hashtag**：5-8 個即可，不需過多

#### LINE OA
- **字數**：不超過三屏手機畫面
- **語氣**：最親切，像一對一聊天，可口語化
- **強調方式**：使用【 】或 ▶ 符號標注重點
- **CTA**：每則訊息必須有明確的下一步（連結按鈕或「回覆此訊息詢問」）
- **Hashtag**：不使用 Hashtag

---

### STEP 4：撰寫兩個版本

**版本一：專業穩重版**
- 強調科學實證、數據背書、國體大合作
- 語氣偏正式但不冷硬
- 適合核心運動員族群、關注成分的消費者

**版本二：感性共鳴版**
- 強調真實生活場景、情感連結、痛點共鳴
- 語氣親近如朋友聊天
- 適合初次接觸品牌、被故事打動的消費者

---

### STEP 5：品牌合規檢查清單

撰寫完成後，逐一確認：

**法規合規（食安法 §28）**
- [ ] 無「治療」、「治癒」、「醫治」等醫療效能詞彙
- [ ] 無「保證」、「神效」、「立刻見效」等誇大不實用語
- [ ] 功效描述使用「輔助恢復」、「營養補給」等合規語境
- [ ] 無未經核實的療效聲稱

**品牌聲音**
- [ ] 使用「你」而非「您」
- [ ] 無「全網最低」、「絕對有效」、「神級」等禁用詞
- [ ] 無否定讀者的措辭（如「你做錯了」）
- [ ] Emoji 用量適中（每段最多 1-2 個）

**平台格式**
- [ ] IG：前三行有 Hook，Hashtag 數量 10-15 個
- [ ] FB：有互動引導問句
- [ ] LINE：不超過三屏，有明確 CTA

**視覺設計建議**
- [ ] 已附上配圖方向建議（素材風格、主色應用比例）

---

### STEP 6：視覺設計搭配建議

每篇貼文附上：
- **建議配圖風格**：自然光微光攝影 / 訓練後真實汗水感 / 產品生活情境
- **主色應用**：背景底色（堅心黑 `#2F2F2F`）+ 強調色（堅心橘 `#ED5D2B`，佔比 10%）
- **文字層級**：主標題（粗體）→ 副標題（中等）→ 正文（一般）
- **避免元素**：過度精修圖片、攝影棚打光感、純黑背景
