# 行銷活動與促銷企劃

## 指令說明
為堅心 Outrun Nutrition 規劃完整的行銷活動或促銷企劃，包含促銷機制設計、推廣時程表、預算分配建議與 KPI 設定，結合耐力運動賽事日曆與年度重要檔期。

## 執行步驟

請提供以下資訊：
- **活動類型**：促銷活動 / 賽事聯動 / 新品上市 / 節日行銷 / 品牌形象活動

- **主要目標**：營收目標 / 新客獲取 / 品牌曝光 / 社群成長
- **可用預算**：（或請系統提供預算分配建議）

### STEP 3：促銷機制設計（2-3 方案）

#### 方案一：系統化購買促進（訂閱制思維）
**適用情境**：鼓勵既有客戶長期使用、建立恢復習慣
**機制設計**：
- 購買「眠易復 + 能易復」組合，享 9 折優惠
- 購買「A.R.G. 完整恢復系統（三品）」，享 85 折 + 免運
- 加入「恢復計畫」每月自動訂購，享固定 88 折
**品牌訴求**：「恢復是一個系統，不是一次性的決定」

#### 方案二：門檻降低轉換（試用體驗）
**適用情境**：吸引新客嘗試，降低高單價購買門檻
**機制設計**：
- 首購優惠：新客 9 折 + 附贈「科學補給指南」電子手冊
- 體驗包裝：能易復 500g 盒裝（較低入門價）+ 購後升級優惠
- 買一送一（成本考量下選擇較低單價品項）
**品牌訴求**：「讓第一次的體感，說服你自己」

#### 方案三：賽事聯動限定（社群驅動）
**適用情境**：配合特定賽事前後，精準觸達高意願消費者
**機制設計**：
- 賽前 2 週：購買 OARC 組合，附贈「賽事補給戰術手冊」
- 賽後 1 週：曬賽事完賽照 tag @outrunnutrition，享恢復組合 9 折
- 配速配套：「全馬 4 小時內」對應不同的補給建議組合
**品牌訴求**：「你的每一次完賽，都值得最好的科學修復」

```
活動名稱：[活動名稱]

  - 社群開始發布相關知識型內容

  - 全通路同步公告（IG / FB / LINE OA / 官網）
  - 廣告切換轉換目標
  - 限時限量資訊公開

活動期間（Day 2 - N-1）
  - 每日社群互動（UGC 徵集、限時動態問答）
  - 廣告持續優化（依 CTR/ROAS 調整預算）
  - KOL 分眾觸達（不同跑者社群）

活動最後 48 小時（倒數）
  - 限時動態倒數提醒
  - 廣告加碼（FOMO 策略）
  - LINE OA 推播最後提醒

活動結束後
  - 成效回顧貼文（感謝參與、分享數據）
  - 未購者再行銷廣告（2 週內）
  - 數據彙整報告
```

### STEP 5：預算分配建議

| 項目 | 佔比 | 說明 |
|------|-----|------|
| 付費廣告（Meta/Google） | 50-60% | 核心轉換工具，依 ROAS 目標調整 |
| KOL / 教練合作 | 15-20% | 口碑觸達，優先選擇有真實賽事經驗者 |
| 素材製作（攝影/設計） | 10-15% | 高品質素材的長期投資回報率高 |
| 平台促銷費用（蝦皮廣告等） | 5-10% | 補充各通路曝光 |
| 備用彈性預算 | 5% | 應對突發高效機會 |

### STEP 6：KPI 設定框架

#### 品牌知名度類指標
- 觸及人數（Reach）目標
- 曝光次數（Impressions）目標
- 社群追蹤人數成長率

#### 互動參與類指標
- 貼文互動率（Engagement Rate）：IG 目標 3% 以上
- 限時動態完看率（Story Completion Rate）：目標 70% 以上
- 留言數量與質量

#### 轉換業績類指標
- 廣告 ROAS（廣告投資報酬率）：目標 3x 以上
- 訂單轉換率（CVR）
- 平均客單價（AOV）
- 新客比例 vs. 回購率

#### 長期品牌類指標
- 會員數成長
- LINE OA 好友數
- 品牌搜尋量（Google Search Console）
